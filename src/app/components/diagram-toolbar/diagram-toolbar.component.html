Keyboard Shortcuts

  • Shift + Click: Select multiple elements simultaneously
  • Ctrl/Cmd + Click: Draw an edge between two nodes
  • Alt + Click: Draw a connector between two nodes
  • Ctrl/Cmd + Z: Undo last action
  • Ctrl/Cmd + A: Select All elements on canvas
  • Ctrl/Cmd + C: Copy selected elements
  • Ctrl/Cmd + V: Paste copied elements
  • Delete/Backspace: Remove selected item(s)
  • Double-click node: Enter inner nested diagram

Mouse Actions

  • Right-click element: Open properties window for configuration
  • Drag node edges: Resize element
  • Drag node centers: Move element around the canvas
  • Drag tendril handle: Draw edge manually
  • Drag canvas: Pan the view across the diagram
