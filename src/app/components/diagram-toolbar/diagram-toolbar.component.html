Keyboard Shortcuts

  • Shift + Click: Multi-select elements for group movement
  • Ctrl/Cmd (Hold) + Click: Hold key down and click start node, then end node to create an edge
  • Alt (Hold) + Click: Hold key down and click start node, then end node to create a connector
  • Ctrl/Cmd + Z: Undo last action (Supports multi-step operations)
  • Ctrl/Cmd + A: Select all elements in the current diagram
  • Ctrl/Cmd + C / V: Copy and Paste selected elements
  • Delete / Backspace: Delete all currently selected items
  • Double-click Node: Drill down into nested diagram architecture

Mouse Actions

  • Right-click: Open Properties Window for colors, labels, and attributes
  • Drag Borders: Resize Nodes, SVGs (aspect-locked), or Bounding Boxes
  • Drag Element: Move individual elements or synchronized selection groups
  • Drag Tendril Handle: Manually draw a connection to another element
  • Drag Canvas: Pan the viewport to navigate large diagrams
  • Zoom Controls: Use the floating strip to adjust diagram scale
